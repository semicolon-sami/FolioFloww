# **App Name**: FolioFlow

## Core Features:

- Home Page: Landing page with intro, tagline, and link to projects.
- Projects Page: Display projects as cards in a responsive grid.
- Project Details: Dedicated subpage for each project with details and links.
- About Page: Bio and interests paragraph with subtle animation.
- Skills Page: Skill grid with frontend, backend, data science, and tools.
- Resume & Links: Downloadable resume and social links with QR code.
- Contact Form: Contact form for submissions.

## Style Guidelines:

- Primary color: Soft indigo (#667eea) for a professional yet modern feel.
- Background color: Off-white (#f9f9f9) to keep focus on the content.
- Accent color: Sky blue (#2ab7ca) for highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font with a modern, machined look suitable for headlines or body text.
- Use FontAwesome or similar for consistent and professional icons.
- Responsive grid layouts using Tailwind CSS for all page elements.
- Subtle fade-in animations for page transitions using Framer Motion.
